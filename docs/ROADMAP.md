# PyTune Box Roadmap

## Completed in v0.1.0-beta
- Project foundation
- Tkinter GUI
- Audio playback (Play, Pause, Stop, Resume, Next, Previous)
- Playlist management (Add, Remove, Clear, Save, Load)
- Folder import
- Metadata display (Title, Artist, Album, Duration)
- Progress tracking
- Basic seek
- Theme system (Light/Dark)
- Settings persistence
- Shuffle mode
- Repeat modes (Off, One, All)
- Keyboard shortcuts
- Simple visualizer
- PyInstaller packaging scripts
- Comprehensive documentation
