# PyTune Box Demo Script

## Demo Duration
Recommended: 60 to 90 seconds

## Demo Flow

1. **Open PyTune Box**
   - Say/show: Desktop music player built with Python and Tkinter.

2. **Open audio files**
   - Show: `File > Open Files`

3. **Play a song**
   - Show: Metadata, duration, progress, visualizer.

4. **Use controls**
   - Show: Pause, resume, stop, next, previous.

5. **Toggle theme**
   - Show: Light/dark mode.

6. **Shuffle and repeat**
   - Show: Shuffle On, Repeat modes.

7. **Save playlist**
   - Show: Save playlist as JSON.

8. **Load playlist**
   - Show: Playlist restored.

9. **Show keyboard shortcuts**
   - Show: Space, arrows, Ctrl+O.

10. **Show packaged EXE**
    - Show: `dist\PyTuneBox\PyTuneBox.exe`

## Recording Tips
- Use OBS Studio or Windows built-in screen recorder.
- Keep the desktop clean and distraction-free.
- Do not show private folders or personal information.
- Use royalty-free or personal audio files to avoid copyright issues.
- Keep the video short and engaging.
