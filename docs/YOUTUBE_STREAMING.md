# YouTube Streaming Integration

## Overview
PyTune Box can stream audio from a YouTube search term or URL directly. This allows users to enjoy online media without saving copyrighted files locally or interrupting the offline file playback.

## Libraries Used
- **yt-dlp**: Extracts direct audio stream URLs and video metadata (title, uploader, duration).
- **python-vlc**: Python bindings to control the VLC Media Player runtime on the host machine.
- **VLC Media Player (64-bit)**: Must be installed separately on Windows for `python-vlc` to locate and load the VLC DLL.

## Why VLC is Used
While the `pygame` mixer is highly efficient for local audio files (such as `.mp3`, `.wav`, and `.ogg`), it lacks native support for network streaming protocols, stream buffering, chunking, and handling dynamic network connections. The VLC runtime provides robust, production-grade network stream buffering and playback, which complements the local player perfectly.

## How It Works
1. **User Input**: The user enters a search term or a YouTube URL into the YouTube Stream field in the GUI.
2. **Asynchronous Extraction**: When the user clicks **Search & Stream**, PyTune Box spins up a background thread that invokes `yt-dlp` to extract the best audio-only format and stream URL. This prevents the Tkinter main event loop from freezing.
3. **VLC Playback**: The direct stream URL is passed to VLC's media player, which handles connection buffering.
4. **Shared Control Logic**: The existing Tkinter UI updates (volume slider, Play/Pause, Stop buttons, visualizer, Now Playing text labels) are seamlessly shared.
5. **Mutual Audio Stop**: When a YouTube stream begins, any playing local song is stopped. Conversely, double-clicking a local playlist item stops the YouTube stream to prevent audio overlap.

## Usage
1. Open PyTune Box.
2. Ensure you have internet access and a 64-bit VLC Player installed on your system.
3. In the **YouTube Stream** section (or via **Tools > YouTube Search & Stream**), type a song query or paste a YouTube URL.
4. Click **Search & Stream**.
5. The status bar will show "Searching YouTube..." and then begin playing.
6. Use the **Play/Pause**, **Stop**, and **Volume** controls as you would for local songs.

## Important Notes
- **Internet Connection**: Internet access is required to search and play YouTube streams.
- **VLC Installed**: VLC Media Player must be installed on the user's computer. If missing, YouTube controls will display a warning dialog and remain inactive, but local music playback will continue to function.
- **No Downloading**: The application does not download, save, or cache YouTube audio files, ensuring full compliance with copyright guidelines and source platform terms.
- **Stream URL Expiration**: Direct stream URLs generated by `yt-dlp` expire after a few hours.

## Troubleshooting
- **VLC missing error**: Ensure that you have installed the 64-bit version of VLC Media Player to match the 64-bit version of Python. Restart your terminal or IDE after installation so python-vlc can find the environment variables.
- **No Internet / Extraction fails**: Ensure your firewall does not block VLC from accessing network ports, and check that the query/URL is valid.
- **Audio overlapping**: Double-clicking a local playlist song will automatically stop the YouTube stream. If both play simultaneously, click **Stop Stream** and restart the track.
