# PyTune Box Project Summary

## Project Type
Python desktop GUI application.

## Purpose
A portfolio-ready desktop music player demonstrating GUI development, file handling, audio playback, metadata processing, JSON persistence, theming, testing, and packaging.

## Core Technologies
- Python
- Tkinter
- pygame
- mutagen
- JSON
- PyInstaller
- pytest

## Major Modules
- **GUI**: Built with Tkinter to manage windows, frames, canvas, and event loops.
- **AudioPlayer**: Wrapper for pygame mixer to control audio state (play, pause, stop).
- **PlaylistManager**: Data structure managing the order of songs, shuffle, and repeat modes.
- **MetadataReader**: Utility using mutagen to extract tags and fall back to filenames when needed.
- **Theme system**: Custom light/dark mode implementation using predefined style dictionaries.
- **Visualizer**: Simulated Tkinter Canvas animation demonstrating continuous updates.
- **Build scripts**: Batch scripts wrapping PyInstaller for reliable `.exe` generation.

## What This Project Demonstrates
- Event-driven GUI programming
- Modular Python architecture
- File system interaction
- Audio playback control
- Error handling
- User settings persistence
- Desktop packaging
- Documentation quality

## Portfolio Value
This project is suitable for a resume or GitHub portfolio because it demonstrates the ability to build a complete, user-facing application from scratch. It highlights not just coding skills, but also architecture design, state management, error handling, documentation, and the final crucial step of packaging the app for end users.
