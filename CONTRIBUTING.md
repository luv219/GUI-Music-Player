# Contributing to PyTune Box

## Current Status
Beta project.

## How to Contribute
1. Fork repository
2. Create feature branch
3. Make changes
4. Run tests
5. Submit pull request

## Development Setup
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Run Tests
```bash
pytest
```

## Coding Guidelines
- Keep code beginner-friendly.
- Keep GUI logic separate where possible.
- Do not break Windows compatibility.
- Do not commit virtual environments.
- Do not commit build/dist folders.
- Add clear comments for complex logic.

## Bug Reports
When submitting a bug report, please include:
- Windows version
- Python version
- Steps to reproduce
- Error message
- Screenshot if useful

## Feature Requests
Use the roadmap as a guide. Please check [docs/ROADMAP.md](docs/ROADMAP.md) before submitting feature requests to see if your idea is already planned.
